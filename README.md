# 🐰 Talking Rabbitt: AI Product Manager Challenge



### 🚀 The Vision

Talking Rabbitt is a conversational intelligence layer that turns complex enterprise data into simple conversations. It is designed for CXOs who need instant insights without navigating legacy dashboards like PowerBI or Tableau.

### 🛠️ Key Features

- **Natural Language Querying:** Ask questions about sales data in plain English.

- **Instant Insights:** Get real-time calculations (e.g., Total Revenue).

- **Executive-First Design:** Focused on speed and simplicity.
